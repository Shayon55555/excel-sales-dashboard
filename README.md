# Sales Performance Excel Dashboard

## Objective
To analyze sales data and build an interactive Excel dashboard for business reporting and decision-making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- KPI Cards
- Charts & Slicers
- GETPIVOTDATA

## Dataset
Superstore Sales Dataset (50,000+ records)

## Key Features
- Dynamic KPIs: Sales, Profit, Orders, Profit Margin
- Interactive slicers for Year, Region, and Category
- Sales trend, regional performance, and category analysis
